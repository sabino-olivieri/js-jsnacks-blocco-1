# ARRAY NUMERI DISPARI (JSnack5)

## Testo della consegna
Crea un array vuoto.
Chiedi per 6 volte all'utente di inserire un numero,
se è dispari inseriscilo nell'array.

## Svolgimento

- [x] creo array vuoto
- [x] creo ciclo
- [x] chiedo numero a utente
    - se è dispari lo inserisco nell'array
- [x] stampo array
